Highlighted Projects

1) Monthly Blood Donation Prediction and Web Deployment

Developed a predictive model to forecast monthly blood donations by state and blood type using an ensemble of regressors (MultipleOutputRegressor, Linear Regression, Decision Tree, Random Forest, Support Vector Regressor, and Gradient Boosting Regressor). Data processing included outlier handling, PCA, and feature engineering. The model was optimized with hyperparameter tuning, and performance was evaluated with RMSE, MAE, and R-squared. Deployed using Flask and Pickle.

View on GitHub | Download ZIP

2) Daily Blood Donation Prediction (Time Series Analysis)

Implemented SARIMAX to predict daily blood donations for four states in Malaysia, incorporating exogenous factors and weekly seasonality. The model was evaluated using AIC, residual analysis (Ljung-Box test, ACF/PACF plots), and forecasted 30-day trends with uncertainty captured via confidence intervals.

View on GitHub | Download ZIP

3) Tweet Dataset Sentiment Analysis and Classification

Performed sentiment analysis on a large tweet dataset using TextBlob for polarity extraction and Logistic Regression for classification. Applied NLP techniques for preprocessing, including tokenization and stopword removal. The model was evaluated using classification reports. Additionally, translated tweets to Malay using GoogleTranslator API.

View PDF | Download ZIP

4) Image Recognition Using CNN (CIFAR-10 Dataset)

Built a CNN model with TensorFlow and Keras to classify images from the CIFAR-10 dataset into 10 categories. The model consisted of convolutional layers for feature extraction and dense layers for classification. Evaluated performance with accuracy metrics and visualized learning over 10 epochs.

View on GitHub | Download ZIP

5) Bank Negara Dataset Analysis: Population Age Distribution and Economic Indicators

This project explores the relationship between population age distribution and economic indicators (GCF, GDS) in Malaysia. Utilized hierarchical clustering to uncover patterns, finding that age dependency ratios negatively affect capital formation and savings, urbanization drives capital formation, and interest rates influence both savings and investment behavior.
